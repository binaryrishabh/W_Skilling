# Assignments

## Assignment 1: Student Management System (Repository Pattern)

**Problem Statement:** A training institute wants to build a Student Management System using ASP.NET Core MVC and Entity Framework Core.

**Requirements:**
- Add new students
- View student details
- Edit student information
- Delete students
- Store data in SQL Server
- Clean separation between Business Logic and Data Access Logic
- Easy testing and maintenance
- Reusable database operations

**Technology:** ASP.NET Core MVC, EF Core, SQL Server, Repository Pattern

## Assignment 2: Product Price Lookup (AJAX)

**Problem Statement:** Create a page where a user can enter a Product ID and click a button. The application must retrieve the Product Price from the server and display only the price on the current page, without triggering a full page reload.

**Requirements:**
- Enter Product ID in input field
- Click button to fetch price
- Retrieve price from server
- Display price without page reload

**Technology:** ASP.NET Core MVC, jQuery AJAX
