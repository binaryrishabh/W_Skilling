## Assignment 2: Product Price Lookup (AJAX)

**Problem Statement:** Create a page where a user can enter a Product ID and click a button. The application must retrieve the Product Price from the server and display only the price on the current page, without triggering a full page reload.

**Requirements:**
- Enter Product ID in input field
- Click button to fetch price
- Retrieve price from server
- Display price without page reload

**Technology:** ASP.NET Core MVC, jQuery AJAX
