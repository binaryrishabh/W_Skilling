# Calculator Unit Testing Assignment

## Assignment Requirements

### User Story 1: Implement Calculator
Create a `Calculator` class with methods:
- `Add(double a, double b)`
- `Subtract(double a, double b)`
- `Multiply(double a, double b)`
- `Divide(double a, double b)` – throws exception on division by zero

### User Story 2: Setup Testing Framework
- Choose a testing framework (NUnit, MSTest, or xUnit)
- Create a test project in same solution
- Write test cases for each method

### User Story 3: Write Unit Tests
Verify:
- Correct results for valid inputs
- Handling of division by zero
- Edge cases (e.g., adding/subtracting zero)
- Use assertions (`Assert.AreEqual`, `Assert.IsTrue`, etc.)

### User Story 4: Test Execution and Reporting
- Run all tests using Test Explorer
- Ensure all tests pass
- Debug and fix any failures
