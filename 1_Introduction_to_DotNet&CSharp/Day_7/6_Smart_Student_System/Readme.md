# Smart Student Data Processing System

## Assignment Overview
	A console-based application that demonstrates Data Structures, File Handling, and OOP concepts in C#. 
	The system efficiently searches, sorts, and manages student data with file-based storage and retrieval.


## 📋 What Was Asked (Requirements)

	### User Stories

		#### User Story 1: Search Student
			- As auser, I want to search a student by ID so that I can quickly retrieve their details
			- Acceptance Criteria:
			  - Implement Linear Search
			  - Implement Binary Search (on sorted data)
			  - Return index or "Not Found"

	#### User Story 2: Sort Student Records
		- As a user, I want to sort student data based on marks so that I can analyze rankings
		- Acceptance Criteria:
		  - Implement at least 2 sorting algorithms
		  - One O(n²) algorithm (Bubble/Selection/Insertion)
		  - One efficient algorithm (Merge/Quick)
		  - Display sorted output

	#### User Story 3: Compare Algorithms
		- As a developer, I want to compare algorithms so that I can choose the best one
		- Acceptance Criteria:
		  - Display Time Complexity
		  - Display Space Complexity
		  - Identify best for small dataset
		  - Identify best for large dataset

	#### User Story 4: File Handling - Create & Write
		- As a user, I want to store student data in a file so that I can persist records
		- Acceptance Criteria:
			- Create a file
			- Write student data into file

	#### User Story 5: File Handling - Append & Read
		- As a user, I want to add new student records and read existing data
		- Acceptance Criteria:
		  - Append new records
		  - Read and display all data

	#### User Story 6: File Handling - Copy & Delete
		- As a user, I want to manage files for backup and cleanup
		- Acceptance Criteria:
		  - Copy file to another location
		  - Delete file

	#### User Story 7: Use OOP Concepts
	- As a developer, I want to design reusable code using OOP concepts
	- Acceptance Criteria:
	  - Use Properties for student attributes
	  - Implement an Indexer to access student list
	  - Add a Custom Attribute (e.g., [StudentInfo])
